编译部署NodeJS

编译部署NodeJS

1. 安装必要的编译软件包

  • 执行命令

    yum install -y gcc make gcc-c++ openssl-devel

2. 升级gcc

  • 2.1 编译nodejs过程中可能会报“ C++ Compiler too old, need g++ 4.9.4 or clang++ 3.4.2 (CXX=g++)”，需要先升级gcc
  • 2.2 通过命令gcc -v查看系统gcc版本号，如果版本号低于4.9.4，则需要升级gcc
  • 2.3 gcc升级过程参考另一篇笔记《GCC升级》

3. 编译部署

  • 将node-v10.15.3.tar.gz安装包放在/usr下，进行解压
    执行命令：tar -zvxf node-v10.15.3.tar.gz

  • 进入源代码所在路径

    cd /usr/node-v10.15.3/
./configure        先执行配合脚本
make               编译,该过程可能会很耗时，起码30分钟
make install       部署

4. 验证

  • 查看版本：node -v
  • 查看版本：npm -v
