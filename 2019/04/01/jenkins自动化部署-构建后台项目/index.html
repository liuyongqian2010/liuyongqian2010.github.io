jenkins自动化部署--构建后台项目

jenkins自动化部署–构建后台项目

1. 简介
  • Jenkins是基于Java开发的一种持续集成工具，用于持续的软件版本发布/测试项目。
2. 部署准备

  • 2.1 maven的安装

    安装执行：yum install maven<br/>
验证版本：mvn -v

  • 2.2 git的安装

    安装执行：yum install git
3. 部署 jenkins

  • 3.1 将 jenkins-2.164.1-1.1.noarch.rpm包放在/usr下，安装执行：sudo rpm -ih jenkins-2.164.1-1.1.noarch.rpm

    安装完毕之后，出现四个文件如下所示：
/usr/lib/jenkins/jenkins.war------WAR包
/etc/sysconfig/jenkins------------配置文件
/var/lib/jenkins/-----------------默认的JENKINS_HOME目录
/var/log/jenkins/jenkins.log------Jenkins日志文件

  • 3.2 启动 jenkins

    启动：sudo service jenkins start
停止：sudo service jenkins stop
查看：systemctl status jenkins.service
<br/>

  • 3.3 启动后在浏览器中输入ip:8080打开jenkins页面，根据提示输入管理员初始密码，该密码一般位于/var/lib/jenkins/secrets/下的initialAdminPassword文件中。

  • 3.4 插件推荐安装完之后进入创建管理员用户的界面，可以使用已有的admin账号密码（密码可以修改）。
4. 赋予 jenkins运行命令的权限
vi /etc/sudoers后增加如下命令：

## Allow root to run any commands anywhere 
root    ALL=(ALL)     ALL
jenkins    ALL=NOPASSWD: ALL

5. 使用 jenkins构建后台项目

  • 5.1 新建任务（输入一个任务名称，如testing001，选择“构建一个自由风格的软件项目”）
  • 5.2 源码管理（选择“Git”，输入仓库URL和git账号）

  • 5.3 修改maven配置文件

    cd /usr/share/maven/conf/
vi settings.xml后修改如下内容：（有两处）

<mirror>
      <id>mirrorId</id>
      <mirrorOf>repositoryId</mirrorOf>
      <name>Human Readable Name for this Mirror.</name>
      <url>http://my.repository.com/repo/path</url>
    </mirror>

<mirror>
      <id>mirrorId</id>
      <mirrorOf>*</mirrorOf>
      <name>Human Readable Name for this Mirror.</name>
      <url>http://192.168.110.67:8090/nexus/content/groups/public/</url>
    </mirror>

  • 5.4 构建

    • 调用顶层Maven目标，目标：clean package -Dmaven.test.skip=true

    • 执行shell，通过脚本进行部署，输入执行shell的命令，如：sudo sh /usr/deploy_cmp.sh。shell脚本可以参考如下所示：

      #!/bin/bash
export BUILD_ID=dontkillme
TOMCATTHREAD=`ps aux | grep 'tomcat-test' | grep -v grep`
if test -z "$TOMCATTHREAD"
then 
    echo "TOMCAT NOT START"
else  
    THREADLIST=(${TOMCATTHREAD// / })
    PID=${THREADLIST[1]}   
    kill -9 $PID   
    sleep 5s
fi  
    cd /usr/tomcat-test/webapps/   
    rm -f cmp.war  
    rm -rf cmp 
    cp /var/lib/jenkins/workspace/testing001/target/cmp.war /usr/tomcat-test/webapps/
    cd /usr/tomcat-test/bin
    ./startup.sh
    sleep 5s
  • 5.5 项目构建位置：/var/lib/jenkins/
