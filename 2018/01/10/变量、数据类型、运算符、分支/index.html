变量、数据类型、运算符、分支

变量

  • 是一个数据存储空间的表示，无需关键字，不需要声明；
  • 命名规则:由数字、字符、下划线组成，不能以数字开始，不能关键字，区分大小写；
  • print()换行，不换行:print(,end=””),用户输入：input(),其中输入的为字符串，可以通过类型转换转为自己需要的类型,可以使用斜杠（ \）将一行的语句分为多行显示;注释：#单行注释 多行注释’’’ ‘’’
  • del删除单个或多个对象的引用
    x=1
x,y,z=1,2,3  
print(x)
del x,y            #删除单个或者多个对象的引用
print("===========================================")
name=input("Please input NAME：")  #input() 键盘录入 类型：str
print("NAME:%s" %name)
age=int(input("Please input AGE：")) #int() 将str转换成int
print("AGE:%d" %age)
work=input("Please input limit：")
print("WORK:%s" %work)
    数据类型
  • 数字Number：int,float,bool,复数
  • 字符串：String
    x=1 #int
y=1.2 #float
z="张三" #字符串
    运算符
  • 算术运算符：+ - /(得到小数), %(取余), //(只得到商，取整),* 幂(返回x的y次幂)
  • 关系运算符：> < >= == <= != 最终的返回bool
  • 逻辑运算符：and or not && || !
  • 成员运算符: in( x 在 y 序列中返回 True) not in( x 不在 y 序列中返回 True)
  • 小练习：请用户输入一个四位数字：求此数字各个位之和
    value=int(input("Please input a 4-bit value:"))
qian=value//1000
bai=value%1000//100
shi=value%100//10
ge=value%10
sum=qian+bai+shi+ge
print(sum)
    分支结构
    score=int(input("Please input score:"))
if(score>=90):
    print("Excellent!")
elif(score>=60):
    print("Medium!")
else:
    print("Flunk!")
