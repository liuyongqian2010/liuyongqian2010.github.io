集合

集合

定义

  • 集合是一个无序不重复元素的集；如果要创建空集合，须用set()

    #第一种方法
s=set()                 #空集合,如果只有一个值，需要加逗号，如s=set('liu',)
s1={'name',2018.01,28,False,(12,22)}

#第二种方法
s2=set({'name',2018.01,28,False,(12,22)})
s2=set(['name',2018.01,28,False,(12,22)])
s2=set(('name',2018.01,28,False,(12,22)))

#注意
s2=set({'name':'Liu','age':23})
print(type(s2))         #result:<class 'set'>
print(s2)               #result:{'name', 'age'}

增删改查

  • add()增加

    s3={1,2,3,4,5}
s4={4,5,6,7,8}
s3.add(999)
print(s3)               #result:{1, 2, 3, 4, 5, 999}

  • remove()删除

    s3.remove(1)
print(s3)               #result:{2, 3, 4, 5}

  • pop()随机返回并删除一个值,没有参数

    s3.pop()
print(s3)               #result:{2, 3, 4, 5}

  • clear()清空

    s3.clear()
print(s3)               #result:set()

并、交、差、对称差集

  • 并集

    print(s3|s4)            #result:{1, 2, 3, 4, 5, 6, 7, 8}

  • 交集

    print(s3&s4)            #result:{4, 5}

  • 差

    print(s3-s4)            #result:{1, 2, 3}

  • 对称差集

    print(s3^s4)            #result:{1, 2, 3, 6, 7, 8}

  • update()将字符串或集合中的数据联合

    s3.update(s4)
print(s3)               #result:{1, 2, 3, 4, 5, 6, 7, 8}

    遍历

    s5=set(('a','b','c','d','e'))
for i in s5:
    print(i,end='')     #result:eacdb

小练习

  • 通讯录管理系统

    print("=======通讯录管理系统=======")
print("1.增加姓名和手机")
print("2.删除姓名")
print("3.修改手机")
print("4.查询所有用户")
print("5.根据姓名查找手机号")
print("6.退出")
print("============================")
User={}
while True:
    i=int(input('请输入操作项：'))
    if i==1:
        name=input('请输入姓名：')
        phone=input('请输入手机：')
        User[name]=phone
    elif i==2:
        name=input('请输入要删除的姓名：')
        if name in User:
            del User[name]
            print('删除成功！')
        else:
            print('姓名不存在！')
    elif i==3:
        name = input('请输入要修改的姓名：')
        if name in User:
            new_phone=input("请输入修改后的手机号：")
            User[name]=new_phone
            print('修改成功！')
        else:
            print('姓名不存在！')
    elif i==4:
        if User is None:
            print('通讯录无记录！')
        else:
            for k,v in User.items():
                print(k,v)
    elif i==5:
        name = input('请输入要查询的姓名：')
        if name in User:
            print(name,'的手机号为',User[name])
        else:
            print('姓名不存在！')
    elif i==6:
         break
    else:
         print('请重新输入操作项！')
