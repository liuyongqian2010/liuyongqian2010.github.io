List类型操作

List类型操作

01

  • lpush key_name value1 value2…将一个或多个值插入到列表头部。 如果 key 不存在，一个空列表会被创建并执行lpush操作

    lpush mylist hello world

  • lpushx key_name value1 value2…将一个值插入到已存在的列表头部，列表不存在时操作无效

  • rpush key_name value1 value2…用于将一个或多个值插入到列表的尾部(最右边)
  • rpushx key_name value1 value2…用于将一个值插入到已存在的列表尾部(最右边)。如果列表不存在，操作无效
  • lpop key移出并获取列表的第一个元素
  • rpop key移出并获取列表的最后一个元素

02

  • lrange key_name start stop将一个值插入到已存在的列表头部，列表不存在时操作无效

  • lrem key_name count value根据参数 COUNT 的值，移除列表中与参数 VALUE 相等的元素

    • count的值可以是以下几种：

      • count > 0 : 从表头开始向表尾搜索，移除与value相等的元素，数量为value
      • count < 0 : 从表尾开始向表头搜索，移除与value相等的元素，数量为value的绝对值

      • count = 0 : 移除表中所有与value相等的值

        lrem mylist -2 hell0

  • rpoplpush用于移除列表的最后一个元素，并将该元素添加到另一个列表并返回

  • brpoplpush list1 another_list timeout从列表中弹出一个值，将弹出的元素插入到另外一个列表中并返回它； 如果列表没有元素会阻塞列表直到等待超时或发现可弹出元素为止

    brpoplpush msg reciver 500 #如在指定时间内没有任何元素被弹出，则返回一个 nil 和等待时长。 反之，返回一个含有两个元素的列表，第一个元素是被弹出元素的值，第二个元素是等待时长

  • blpop list1 list2…timeout移出并获取列表的第一个元素， 如果列表没有元素会阻塞列表直到等待超时或发现可弹出元素为止

    blpop list1 100          #如果列表为空，返回一个 nil 。 
                        否则，返回一个含有两个元素的列表，第一个元素是被弹出元素所属的 key ，
                        第二个元素是被弹出元素的值

  • brpop list1 list2…timeout移出并获取列表的最后一个元素， 如果列表没有元素会阻塞列表直到等待超时或发现可弹出元素为止

    brpop list 100           #假如在指定时间内没有任何元素被弹出，则返回一个 nil 和等待时长。         
                        反之，返回一个含有两个元素的列表，第一个元素是被弹出元素所属的 key ，
                            第二个元素是被弹出元素的值

03

  • lset key_name index value过索引来设置元素的值
  • ltrim key_name start stop 对一个列表进行修剪(trim)，让列表只保留指定区间内的元素

  • lindex kye_name index用于通过索引获取列表中的元素

    lpush mylist hello world
lindex mylist 1        #列表中下标为指定索引值的元素

  • linsert key_name before exist_value new_value用于在列表的元素前或者后插入元素。 当指定元素不存在于列表中时，不执行任何操作

    linsert mylist before hello 2018

  • llen key_name用于返回列表的长度
