Set类型操作

Set类型操作

01

  • 1.sadd key_name value1 value2 …将一个或多个成员元素加入到集合中，已经存在于集合的成员元素将被忽略

    sadd myfamily yongmei yongqin yongxing

  • 2.scard key_name返回集合中元素的数量

    scard myfamily    #返回集合的数量。 当集合 key 不存在时，返回 0

  • 3.sismember key_name value判断成员元素是否是集合的成员

    sismember key4 a  #result:1

  • 4.smembers key_name返回集合中的所有的成员。 不存在的集合 key 被视为空集合

    smembers key4      #result:a c

02

  • 5.sdiff first_key other_key1 other_key2 …返回给定集合之间的差集。不存在的集合 key 将视为空集,差集的结果来自前面的first_key

    sadd key1 a b c d
sadd key2 a c
sdiff key1 key2     #result:b d

  • 6.sdiffstore key_name key1 key2将给定集合之间的差集存储在指定的集合中。如果指定的集合 key 已存在，则会被覆盖

    sdiffstore key3 key1 key2
smembers key3      #result:b d

  • 7.sinter key1 key2 返回所有给定集合的交集

    sinter key1 key2   #result:a c

  • 8.sinterstore key_name key1 key2将给定集合之间的交集存储在指定的集合中。如果指定的集合已经存在，则将其覆盖

    sinterstore key4 key1  key2
smembers key4      #result:a c

  • 9.sunion key1 key2 …返回给定集合的并集。不存在的集合 key 被视为空集

  • 10.sunionstore destination key1 key2 …将给定集合的并集存储在指定的集合 destination 中。如果 destination 已经存在，则将其覆盖

03

  • 11.smove source destination member将指定成员 member 元素从 source 集合移动到 destination集合,此为原子性操作

    smove key3 key4 b
smembers key4      #result:a b c

  • 12.spop key_name 用于移除并返回集合中的一个随机元素

    spop key4          #返回b

  • 13.srem key_name member1 member2 …用于移除集合中的一个或多个成员元素，不存在的成员元素会被忽略

    screm key4 a c     #返回被成功移除的元素的数量

  • 14.srandmember key [count] 用于返回集合中的一个随机元素

    srandmember key1 2  #result:b d
