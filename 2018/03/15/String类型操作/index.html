String类型操作

String类型操作

01

  • 1.set key value设置指定key的值

    set myname liuyongqian
set age 108

  • 2.get key获取指定key的值

    get myname

  • 3.getset key_name value用于设置指定key的值，并返回key的旧值，当key 没有旧值返回 nil

    getset myname lailai     #result:'liuyongqian'

  • 4.getrange key_name start end获取存储在指定key中字符串的子字符串

    set myname liuyongqian
getrange myname 0,2      #result:'liu'

  • 5.setrange key_name offset value用指定的字符串覆盖给定 key 所储存的字符串值，覆盖的位置从偏移量offset开始

    setrange job 5 manager
get job               #result:code-manager

  • 6.strlen key_name 用于获取指定 key 所储存的字符串值的长度。当 key 储存的不是字符串值时，返回一个错误

    strlen job             #result:12

02

  • 7.mset key1 value1 key2 value2 …用于同时设置一个或多个 key-value 对

    mset k1 hello k2 world     #result:OK

  • 8.setnx key_name value在指定的key不存在时，为key设置指定的值；若key存在，则设置失败，覆盖也失败

    exists job            #result:0
setnx job code-farmer #result:1

  • 9.msetnx key1 value1 key2 value2 …用于所有给定 key 都不存在时，同时设置一个或多个 key-value 对,若某个键已存在，则设置失败

  • 10.mget key1 key2 … 返回所有(一个或多个)给定 key 的值。 如果给定的 key 里面，有某个 key 不存在，那么这个 key 返回特殊值 nil 

    mget myname age         #result:'lailai''108'

03

  • 11.getbit keyname offset对key所储存的字符串值，获取指定偏移量上的位(bit),当偏移量offset比字符串值的长度大，或者key不存在时，返回0 

    set mynum a  #'a'的ASCII码是 97,转换为二进制是：1100001。offset的学名叫做“偏移” 。
             #二进制中的每一位就是offset值，比如在这里ffset 1等于'1' ,offset2等于'1',offset 7 等于'1' ，offset是从左往右计数的
getbit mynum 2           #result:1

  • 12.setbit key_name offset value 用于对key所储存的字符串值，设置或清除指定偏移量上的位(bit)

    #将mynum的值a变成b，也就是将01100001变成01100010（b的ASCII码是98），
#即将'a'中的offset 6从0变成1，将offset 7从1变成0
setbit mynum 6 1
setbit mynum 7 0
get mynum            #result:'b'

04

  • 13.setex key_name timeout value为指定的key设置值及其过期时间，单位为秒。如果 key 已经存在，将会替换旧的值

    setex mymemory 8 loveyou         #待8秒后mymemory的值为空

  • 14.psetex key_name timeout value以毫秒为单位设置 key 的生存时间

05

  • 15.incr key_name将key中储存的数字值增一
    如果key不存在，那么 key的值会先被初始化 0，然后再执行incr操作
    如果值包含错误的类型，或字符串类型的值不能表示为数字，那么返回一个错误

    set page 108
incr page
get page          #result:109

  • 16.incrby key_name value将key中储存的数字加上指定的增量值

    set rank 110
incrby rank 50
get rank          #result:160

  • 17.incrbyfloat key_name value为key中所储存的值加上指定的浮点数增量值

    set price 3.14
incrbyfloat price 0.01
get price         #result:3.15

  • 18.decr key_name将key中储存的数字值减一

  • 19.decrby key_name value将key所储存的值减去指定的减量值

  • 20.append key_name new_value 用于为指定的key追加值

    如果 key 已经存在并且是一个字符串，append命令将value追加到 key 原来的值的末尾
    如果key不存在，append就简单地将给定key设为 value ，就像执行set key value 一样

    exists myname       #result:0
append myname  liu
append myname  yongqian
get myname          #result:liuyongqian
