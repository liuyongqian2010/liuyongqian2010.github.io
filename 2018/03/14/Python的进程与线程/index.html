Python的进程与线程

Python的进程与线程

概念

  • 进程：独立（内存独立，cpu使用独立）的所有子任务的集合，启动进程开销大，进程之间很难共享数据和数据通信
  • 线程：进程中的每个子任务，不能独立存在，cpu执行的最小单位；依赖进程（内存共享）启动开销小，线程之间共享数据容易

  • 实现线程的两种形式

    • 函数方式实现：线程必须依赖函数实现，不能单独运行。单函数结束，线程结束。如果想让线程一致运行，就要想办法让程序不结束，比如在程序最后加一个input()或者死循环

      import threading,time
thr=threading.current_thread()  #threading.current_thread()返回当前的线程变量
thr.setName('主线程')
print('当前线程：',thr.getName())
def fun1(t_name):
    for i in range(1,6):
        print(t_name,i)
        time.sleep(2)
threading._start_new_thread(fun1,('子线程',))   #第二个参数类型为元组，为第一个参数传参
for i in range(1,6):
    print(thr.name,i)
    time.sleep(2)

      • a继承threading.Thread
      • b写构造方法，且必须调用父类的构造方法
      • c重写父类的run方法，会在start之后自动调用

      • d实现开始方法，如果重写了start()方法，一定要调用父类的start()

thread1=myThread()

资源共享引发的问题—锁

  • 由于并发的问题，需要加锁
  • 锁有锁定和未锁定两种状态，当一个线程要访问共享数据时，必须要先获得锁定，如果已经有别的线程获得锁定，那么就进入暂停状态，等别的线程把所释放后再进行操作

  • Conditon：更精确的控制锁，提供了四个方法，上锁-acquire()，等待-wait()，解锁-release()，唤醒-notify()

    import threading,time
sum=20  #总票数
count=0  #当前抢到票数
lock=threading.Lock()             #建锁
class qiangpiao(threading.Thread):
    def __init__(self,t_name):
        threading.Thread.__init__(self)
        self.name=t_name
    def run(self):
        global sum, count
        while True:
            lock.acquire()   #上锁
            if sum<=0:
                return
            else:
                sum-=1
                count+= 1
                print(self.name,'抢到了第{0}张票，剩余{1}张票'.format(count,sum))
            lock.release()    #释放
            time.sleep(1)
chuangkou=qiangpiao('窗口')
huangniu=qiangpiao('黄牛')
phone=qiangpiao('手机')
chuangkou.start()
huangniu.start()
phone.start()

多线程互相影响

  • 厨师与吃货

    import threading,time
zhenglong=[]      #蒸笼，存包子资源
class Cook(threading.Thread):
    def run(self):
        while True:
            chi_cond.acquire()
            global zhenglong
            if len(zhenglong)==0:
                print('厨师开始蒸包子')
                for i in range(1,8):
                    print('厨师正在蒸第{0}个包子'.format(i))
                    zhenglong.append(i)
                    time.sleep(1)
                print('包子蒸完，唤醒吃货......')
                chi_cond.notify()
            chi_cond.release()
            zheng_cond.acquire()
            zheng_cond.wait()
            zheng_cond.release()
class Chihuo(threading.Thread):
    def __init__(self,c_name):
        threading.Thread.__init__(self)
        self.name=c_name
    def run(self):
        while True:
            chi_cond.acquire()
            global zhenglong
            if len(zhenglong) == 0:
                zheng_cond.acquire()
                zheng_cond.notify()
                zheng_cond.release()
                chi_cond.wait()
            else:
                baozi=zhenglong.pop()
                print(self.name,'吃了第{0}个包子，还剩{1}个包子'.format(baozi,len(zhenglong)))
                time.sleep(1)
            chi_cond.release()
chi=threading.Lock()
chi_cond=threading.Condition(lock=chi)
zheng=threading.Lock()
zheng_cond=threading.Condition(lock=zheng)

cook=Cook()
aa=Chihuo('aa')
bb=Chihuo('bb')
cc=Chihuo('cc')
cook.start()
aa.start()
bb.start()
cc.start()
