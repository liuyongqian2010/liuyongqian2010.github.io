DML和DQL

DML和DQL

DML

  • 存储引擎

    • 存储引擎的类型：MyISAM、InnoDB、Memory、CSV等9种

      • 默认InnoDB，支持事务处理、数据行锁定、外键约束，表空间较大
      • MyISAM，支持全文索引，表空间较小，以查询访问为主
      • 查看当前默认存储引擎show variables like 'storage_engine%'
      • 可以在配置文件my.ini中修改存储引擎default-storage-engine=?

      • 设置表的存储引擎

        create table 表名(
    省略代码
)engine=存储引擎
    • MyISAM类型表文件
      • *.frm：表结构定义文件
      • *.MYD：数据文件
      • *.MYI：索引文件
    • InnoDB类型表文件
      • *.frm：表结构定义文件
      • ibdata1文件
    • 存储位置，可查my.ini中的datadir

  • DML语句

    • 插入单条数据记录

      insert into 表名(字段1,字段2,字段3...) values(值1,值2,值3...)
#如果插入所有字段可省略字段名，如果是部分数据，字段名与值需要一一对应

    • 插入多条数据记录

      insert into 表名(字段1,字段2,字段3...) values(值1,值2,...),(值3,值4,...),(值5,值6,...)

    • 将查询结果插入新表

      create table 新表(        #自动创建一个新表
    select 字段1,字段2 from 表名
)

    • 更新数据记录

      update 表名 set 字段1=值1,字段2=值2,... where 条件

    • 删除数据记录

      delete from 表名 where 条件
truncate table 表名   #该语句删除后将重置自增列，表结构及其字段、约束、索引保持不变，执行速度比delete语句快

DQL

  • 什么是查询
    查询产生一个虚拟表，看到的是表形式显示的结果，但结果并不真正存储，每次执行查询只是从数据表中提取数据，并按照表的形式显示出来

    SELECT    <列名|表达式|函数|常量> 
FROM      <表名> 
[WHERE    <查询条件表达式>] 
[ORDER BY <排序的列名>[ASC或DESC]]

select studentNo as '学生编号',studentName as '学生姓名'
from student 
where sex='男' and gradeID=1 
order by studentNo desc

  • 聚合函数

    avg() 返回某字段的平均值
count()返回某字段的行数
max()返回某字段的最大值
min()返回某字段的最小值
sum()返回某字段的和

select avg(studentResult) from result

  • 字符串函数

    字符串连接concat(str1,str2,...)
select CONCAT('Liu','Yong','qian',2018)  #result:LiuYongqian2018

字符串替换insert(str,start,length,newstr)
select INSERT('ABCDEFGHIJK',3,5,'9')     #result:AB9HIJK

将字符串转为小写lower(str)
select lower('LIUYONGQIAN')                #result:liuyongqian

将字符串转为大写upper(str)
select upper('liuyongqian')                #result:LIUYONGQIAN

字符串截取substring(str,start,length)
select SUBSTRING('LiuYongqian',4,4)      #result:Yong

  • 时间日期函数

    curdate()获取当前日期
select curdate()      #result:2018-01-01

curtime() 获取当前时间
select curtime()      #result:12:12:12

now()获取当前日期和时间
select now()          #result:2018-03-01 12:44:24

week(date) 返回日期date为一年中的第几周
select week(now())    #result:2

year(date)返回日期date的年份
select year(now())     #result:2018

hour(time)返回时间time的小时值
select hour(now())     #result:19

minute(time)返回时间time的分钟值
select minute(now())   #result:48

datediff(date1,date2)返回参数之间相隔的天数
select datediff(now(),'2008-8-8')  #result:3493

adddate(date，n)计算日期date加上n天后的日期
select adddate(now(),5)  #result:2018-03-30 19:48:44

  • 数学函数

    ceil(x)返回大于或等于数值x的最小整数 
select ceil(2.3)      #result:3

floor(x)返回小于或等于数值x的最大整数
select floor(2.3)     #result:2

rand()返回0-1之间的随机数
select rand()         #result:0.8155794216465013
