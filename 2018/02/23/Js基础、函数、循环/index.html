Js基础、函数、循环

基础

  • js是一种广泛应用于客户端网页开发的脚本语言，是基于对象的弱类型语言，无需编译，由JavaScript引擎解释执行
    其核心部分只包括基本的语法构造（比如操作符、控制结构、语法）和标准库（一系列具有各种功能的对象如Maths、Date等）
    完整的js包括三部分：核心（ECMAScript)、BOM、DOM
  • js数据类型
    基本数据类型包括：原始类型：string、number、boolean、undefined、bull，引用类型：object、function、date、number、string、boolean

  • 运用js的三种方式：
    第一种，要执行的脚本语句直接编写在body中，如：

    <body>
    <button onclick="console.log('hello world')">打印</button>
</body>

    第二种，外部，内部脚本块中可以执行多条语句：

    <script>
    document.write('<b>你好</b>');
    console.log('脚本执行完成');
</script>

    第三种，将JavaScript代码写入一个后缀名为js的文件中并在中引入：

    <script src="js文件.js"></script>

  • js语法规范

    语句中大小写敏感，使用分号或者换行结束；

    单行注释：// 多行注释：/ /

    基本输出语句：

    console.log();//控制台输出
document.write();//页面输出
window.alert();//弹出提示框
  • 变量
    存储信息的容器，使用关键字var，用=为变量赋值，没有初始化的变量自动取值为underfined；一条语句中可声明多个变量，如：var name1,name2,name3=liu,
    不允许使用语言关键字和保留字变量名

    标准关键字:
    break，case，catch，continue，deflault，delete，do，else，false，finally，or，function，if，in，instanceof，new，null，return，switch，this，throw，true，try，typeof，var，with，undefined

    标识符命名规范：可以包含字母、数字、下划线、美元符号；不能以数字开头
  • 常量
    是值不能改变的量，使用const关键字声明变量，一般常量名使用纯大写形式，如 const PI=3.15

函数

  • 运算符

    算数运算：加（+）、减（-）、乘（*）除（/）求余（%）
关系运算：“>”(大于)，“<”(小于)，“>=”(大于等于)，“<=”(小于等于)，“==”(等于)，“!=”(不等于)，===（类型、数值全等），!==（类型、数值全不等）
逻辑运算：&&（与）、||（或）、!（非）
赋值运算：+=（先运算后赋值） =+（先赋值后运算）   -=   *=   /=   %=
字符连接： +

三目运算：表达式1？表达式2：表达式3  举例：var age = 20；  var msg = age>18?”成年人”:”未成年”

  • 数据类型转换函数

    Number()
把一个string解析为number，如果包含非法字符，则返回NaN
parseFloat()
解析出一个string或number的浮点数部分，如果没有可转换的部分，则返回NaN
parseInt()
解析出一个string或number的整数部分，如果没有可转换的部分，则返回NaN
toString()
转换成字符串，所有数据类型均可转换为string类型

  • 函数

    • 函数是一段定义好，并可以被反复使用的代码块，其中以包含多条可执行语句；函数本质上是功能完整的对象

    • 如

      function add(ename,epwd){
  console.log('用户名'+ename+'密码'+epwd);
 }
add('Marry','123')

    • ECMAScript中预定义全局函数，可直接应用，如：
      encodeURI函数用于对统一资源标识符（URI）进行编码，返回编码后的URI字符串
      decodeURI函数用于对使用encodeURI编码之后的字符串进行解码，举例：

      var ur=”http://www.baidu.com/name/web/前端”;
var encoded = encodeURI(ur);
var decoded = decodeURI(encoded);

分支结构

  • if语句
  • if-else语句

  • switch-case语句

    <body>
<button onclick="fn3()">Switch</button>
<script>
function fn3(){
    var m1=Number(prompt("Please input a value:"));
    switch (m1)
    {
        case 1:
            console.log("游戏人生");
            break;
        case 2:
            console.log("奋斗人生");
            break;
    }
}
</script>
</body>

  • switch-case在实现分支功能时和if–else的主要区别在于：

    if…else…可以判定相等或不等的情况，适用性广

    switch…case…结构更清晰、效率更高;但是一般只用于指定变量相等于某个范围内的某个特定的值

循环结构

  • while语句

    var i = 0;
while(i<10){
    console.log(i);
    i++;            //循环条件每执行一次就改变一次
}

  • do-while语句

    do{
    可执行语句;
}while(boolean表达式);
  • while和do-while的区别

    while循环 “先判断再执行”

    do-while循环 “先执行再判断”

    当初始情况不满足循环条件时，while循环一次都不会执行；do-while循环不管任何情况至少执行一次。

  • for 语句

    for(表达式1;表达式2;表达式3){
    循环体语句;
}
  • break用于循环，可使程序终止循环而执行循环后面的语句，常常与条件语句一起使用
  • continue关键字只能用于循环中，作用为跳过本次循环体中剩余语句而执行下一次循环
