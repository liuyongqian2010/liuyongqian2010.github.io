标签、链接、列表、框架

标签、链接、列表、框架

标签

  • 文本格式化标签
    格式化标签是使用一些标签来改变字体的形态和式样，以便产生一些浏览者习惯的或约定的显示效果，放在标签之间的文字受其控制

    <b>            定义粗体文本     
<i>            定义斜体字
<del>        定义删除字       
<sub>        定义下标字
<sup>        定义上标字       
<small>     定义小号字
<strong>    定义加重语气字    
<em>        定义着重文
<ins>        定义插入字

  • 引用输出标签

    <code>            定义计算机代码        
<pre>            定义预格式文本
<address>        定义地址          
<q>                  定义短的引用
<bdo dir='rtl'>    定义文字方向

  • 字符实体
    在 HTML 文档中，有些字符没办法直接显示出来，需要用一些代码来表示它们

    空格      &nbsp
<        &lt                
>        &gt
&        &amp            
"        &quot
®        &reg            
TM         &trade
Ⓒ        &copy            
×        &times
÷        &divide

  • html常用标签

    <!DOCTYPE html/>
<html>
<head lang="en">
    <meta charset="utf-8"/gb2312>      
    <meta name="Author" content="Liu"> #<meta> 定义网页的基本信息（供搜索引擎抓取）
    <title>document</title>            #定义网页标题（唯一）
    <link ...>                         #导入外部文件
    <script>...</script>               #定义JS脚本语言
    <style>...</style>                 #定义CSS样式
</head>
    <body>
          <hn>...</hn>                   #标题标签
        <p>...</p>                     #段落标签
        <br/>                          #换行标签
        <hr/>                          #水平线标签
        <span>...</span>               #无语义标签
        <div>...</div>
        <img src='图片路径'>            #图片标签
        <!--注释内容-->                 #注释
    </body>
</html>
  • 元素归纳
    • 块状元素（独占一行，可设置宽高）：address blockquote center div ul ol dl form hn hr p pre table
    • 内联元素（只可设置水平方向的margin padding）：a b br code em font i img input label q select small span strong sub sup textarea u下划线 strike中划线

链接

  • 超链接

    #超链接语法
<a href="资源地址" target="窗口名称" title="鼠标指向连接显示的文字">超链接名称</a>
注：target,  _blank在新窗口打开    _self在同一个窗口打开    _top在浏览器的整个窗口中打开

<a href="基本标签.html" title="这是跳转基本标签的超链接" target="_blank">这是第一个网页</a>
<a href="pictures/大山.jpg" >这里是下载图片的链接</a>

列表

  • 无序列表

    <ul type="square">
    <li>小方块</li>
</ul>
<ul type="circle">
    <li>空心圆</li>
</ul>
<ul type="disc">
    <li>默认实心圆</li>
</ul>

  • 有序列表
    顺序编号的设置是由两个属性type和start 来完成的

    #语法：
<ol type=A  start=A>           #type用于编号的类型，可以为数字、大小写字母、大小写罗马数字
    <li>......</li> 
    <li  value="n">....</li>   #value 改变列表行项目的特定编号
</ol>

<ol type=1 start=3>
    <li>仓库</li>
    <li>仓库</li>
    <li>仓库</li>
</ol>

  • 嵌套列表

    <ol type=A>
    <li>名次</li>
    <ul>
        <li type="circle">姓名</li>
    </ul>
</ol>

  • 自定义列表

    #默认两个层次，第一层为列表项标签<dt>,第二层为注释项标签<dd>
<dl>
    <dt>第一项</dt><dd>注释一</dd>
    <dt>第二项<dd>注释二
    <dt>第三项<dd>注释三
</dl>

框架

  • 框架就是把一个浏览器窗口划分为若干个小窗口，每个窗口可以显示不同的 URL 网页。使用框架可以非常方便的在浏览器中同时浏览不同的页面效果，也可以非常方便的完成导航工作
    而所有的框架标记要放在一个html文档中。html页面的文档体标签被框架集标签frameset所取代，然后通过frameset的子窗口标签frame定义每一个子窗口和子窗口的页面属性

  • 框架集frameset控制

    • 框架结构可以根据框架集标签frameset的分割属性分为3种：左右分割、上下分割、嵌套分割

      border                设置边框粗细，默认是5px
bordercolor            设置边框颜色
frameborder            指定是否显示边框，0表示不显示，1表示显示
cols                可用像素、百分比
rows                可用像素、百分比
framespacing="5"    表示框架间保留空白的距离

  • 子窗口frame标签的设定

    • frame是个单标签,frame标签要放在框架集frameset中,frameset设置了几个子窗口就必须对应几个frame标签，而且每一个frame标签内还必须设定一个网页文件src=”*.html”,其常用属性有:

      src                指定加载的URL文件地址
bordercolor        设定边框颜色
frameborder        指定是否需要边框，1显示边框，0不显示
border            设置边框粗细
name            指示框架名称，是连接标记的target所要的参数
scorlling        指示是否需要滚动条，auto yes no
marginwidth        设置内容与窗口左右边缘的距离，默认为1
marginheight    设置内容与窗口上下边缘的距离，默认为1
width            框窗的宽及高，默认width-100，height-100
align            等于left right center top middle bottom

  • 比如

    <!doctype html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>框架</title>
    </head>
    <frameset cols="25%,50%,25%">
        <frame src="基本标签.html">
        <frame src="列表表格.html">
        <frame src="......">
        <noframes>                #这对标签的作用是避免浏览器版本太旧而无法显示时会弹出提示语，而不是一片空白
            <body>您的浏览器无法处理框架</body>
        </noframes>    
    </frameset>
</html>
