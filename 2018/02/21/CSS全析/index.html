CSS全析

CSS基础、样式、新增

定义

  • CSS是层叠样式表，即多重样式定义被层叠在一起成为一个整体，在页面设置中是标准的布局语言，用来控制元素的尺寸、颜色和排版
    用来将“页面结构代码”和“网页格式风格代码”分离开。从而使页面设计者可以对页面的布局进行更多地控制

选择器

  • 通用选择器*{color:red}
  • 分组选择器h1,h2,h3,h4,p{color:red}
  • 后代(派生)选择器p span{color:red} 只针对p标签中的span标签有用
  • 子选择器p>span{border:1px solid red} 只针对p标签中的第一级span标签有用
  • 兄弟选择器h1+p {color:red} h1标签后的第一个p标签有用，并且需要时同级且同一个父级
  • 标签选择器，p、span、div等，如p{color：#fff}
  • 类选择器，如 .Liu{color：#f9f9f9}
  • ID选择器，如 #Liu{color：blue}
  • 优先级：ID选择符、类选择符、HTML标记选择器。可用!important提升样式表的优先权，如： .p{clolor:#ff0000!important}

引入

  • 内联，<p style="color:red"></p>
  • 内部，将CSS代码添加到头部的style标签中
  • 外部，也是头部<link rel="stylesheet" type="text/css href="css文件路径">

  • 图片样式

border-style:none无 solid实线 dashed虚线 dotted点划线  double双线

3.定义对齐方式

  • 背景样式

    1.设置页面背景图

2.设置背景图重复样式
background-repeat：repeat-x   repeat-y  no-repeat

3.定义背景图片位置
background-position：center  水平居中

                     top   left   图片在页面顶端且局左
                     center  left  图片在左侧水平居中

4.固定背景图片
background-attachment：scroll  背景图片随对象内容滚动
                       fixed   背景图片不随对象内容移动

5.设置背景颜色
background-color：颜色参数

盒子模型

  • content（内容）：可以是文字、图片等元素
  • padding（空白）：也成内边距，补白，用来设置盒子模型的内容与边框之间的距离
  • border（边框），即盒子本身，该属性可设置内容边框线的粗细、颜色和样式等
  • margin（边界），也称外边距，用来设置内容与内容之间的距离

元素定位

  • 在CSS中，可以通过position、float、z-index等定位元素进行定位
    • position{position:relative|absolute|static|fixed}
      • relative相对定位:相对于原来的位置，保留原有空间
      • absolute绝对定位:脱离文档流,选取其最近的父级定位元素定位，当父元素的position为static时，该元素将以body坐标原点进行定位;一般情况下绝对定位配合相对定位
      • static所有元素定位的默认值
      • fixed该参数固定的对象是可视窗口而并非body或父级元素
    • 相对定位
      • 通过某个元素的上下左右移动来对其进行相对地定位,是相对于原来位置的定位，但仍占据原来位置,因此，移动元素会造成覆盖其他框
    • 绝对定位
      • 通过某个元素的上下左右移动来对其进行定位,默认依据父级的坐标原点进行定位,如果父级position属性为默认值，且top、left、bottom、right4个值均未设置，则是参照浏览器的左上角
    • float
      • { float:left|right|none } 水平方向定位
      • { clear:left|right|both } 清除浮动
    • z-index
      • CSS中可以处理元素的高度、宽度、深度三个维度，深度的处理就是z-index

块元素、内联元素

  • 块元素
    • display:block总是在新行上开始，行高以及顶底边距可以控制
    • display:inline和其他元素都在一行上，行高及顶和底边距不可改变
    • display:inline-block和其他元素都在一行上，元素的高度、宽度、行高以及顶和底边距都可设置

新增

  • 文字阴影
    text-shadow:X轴 Y轴 模糊距离 颜色（如果不需要阴影模糊效果，可以blur值设置为0）
  • 文字描边
    text-stroke:宽度值 颜色值
  • 文本溢出
    overflow：hidden；overflow-X:hidden
  • 渐变
    默认从上到下：background:linear-gradient(red,blue);
    从左往右：background:linear-gradient(to right,red,blue)

  • 动画

    第一步：定义动画，

    @keyframes 动画名{
 0%{left：0px}
 100%{left：100px}
}

    第二步：调用动画，
    animation：动画名 持续时间 播放方式 播放次数；（animation: c 5s linear infinite;）
