Github+Hexo博客更换主题

1.挑选主题

  • 1.1在Themes|Hexo中挑选自己中意的主题，记住名字，比如“Again”
  • 1.2在gitbash中cd到博客所在文件夹的位置如”cd /e/github/myblog”,继续输入”https://github.com/lyyourc/hexo-theme-again"可克隆最新的Again主题到你的博客文件夹里面
  • 1.3克隆之后可在你的博客文件中的themes下看到刚刚下载的Again主题

2.安装主题

  • 2.1修改站点配置文件（你的博客最底层目录下的_config.yml），用任一编辑器打开找到“theme”选项，将原先的主题名称删除改为Again即可，注意theme的冒号与Again之间有空格

  • 2.1.1这里可以按照Again的官方指导教程，配置一下Site模块中的内容：

  • 2.2在gitbash中输入”hexo s –debug”启动hexo服务器进行本地测试一下，在浏览器打开“http://localhost:4000/”

  • 2.3修改主题配置文件（你的博客目录下的themes/Again下的_config.yml）,找到scheme关键字，保证只有一个scheme前面没有#号，说明这个scheme选中

3.发布主题

  • 在gitbash中依次输入”hexo clean” “hexo generate” “hexo deploy”，字幕滚动完后，新博客就可以访问了
