同时安装Python2与python3

由于平时学习中需要两种版本不时的替换，索性通过网上查找资料自己配置安装了两个版本，以下是小白整理的一些步骤。

Python2安装

  • 1.1到Python官网下载Python2安装文件，如：python-2.7.12.amd64.exe
  • 1.2点击”python-2.7.12.amd64”进行安装后将该安装目录下的python.exe文件重新命名为python2.exe
  • 1.3在电脑–属性–高级系统设置–高级–环境变量–系统变量–Path–编辑中添加环境变量，如”C:\Python27”，即python2的安装目录
  • 1.4在Dos窗口中输入python2如出现python2的版本信息即表明安装成功
  • 1.5到Pip官网下载名称类似”setuptools-28.7.0.zip”的pip依赖包，在cmd中cd到该依赖包解压后的目录下，输入python2 setup.py install
  • 1.6到Pip官网下载名称类似”pip-8.1.2.tar”的文件并解压，在cmd中cd到该文件解压后的目录下，再次输入python2 setup.py install
  • 1.7在cmd中输入pip2,如果出现”Usage”字样说明安装成功
  • 1.8测试使用：在cmd中输入pip2 install selenium,如果窗口命令哗哗的下滚，说明可以了；也可下载python其他包，用cmd进入到其解压后的目录下，输入python2 setup.py install

Python3安装

  • 1.1到Python官网下载Python3安装文件，如：python-3.5.0-amd64.exe
  • 1.2点击”python-3.5.0-amd64.exe”进行安装后将该安装目录下的python.exe文件重新命名为python2.exe
  • 1.3在电脑–属性–高级系统设置–高级–环境变量–系统变量–Path–编辑中添加环境变量，如”C:\Python35”，即python3的安装目录
  • 1.4在Dos窗口中输入python如出现python3.5的版本信息即表明安装成功；由于未修改python3安装目录下的python.exe文件名，此处python就是python3
  • 1.5在cmd中输入pip3,如果出现”Usage”字样说明自动安装成功；python3自带pip，无需手动安装
  • 1.6测试使用：在cmd中输入pip3 install selenium,如果窗口命令哗哗的下滚，说明可以了；也可下载python其他包，用cmd进入到其解压后的目录下，输入python setup.py install
  • 该文参考大神的佳作在同一台电脑上同时安装Python2和Python3
