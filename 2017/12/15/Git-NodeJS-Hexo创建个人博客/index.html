Git+NodeJS+Hexo创建个人博客

##1.Git篇

  • Windows环境下，安装（Git-2.15.1.2-64-bit.exe）后可打开gitbash运用”git version”查看版本

##2.NodeJS篇

  • Windows环境下，安装（node-v8.9.3-x64.msi）并配置环境变量后，在win-dos窗口中运用”node -v”查看版本

##3.Hexo篇（接下来均在gitbash中完成

  • 3.1手动创建一个将来存放blog的文件夹，在gitbash中cd到该文件夹下运用”npm i -g hexo”安装hexo后使用”hexo -v”查看版本；

  • 3.2运用”hexo init”初始化文件夹

##4.关联github

  • 4.1在github网页端创建自己的账号后new一个repository，name如liuyongqian2010.github.io，记得勾选initialize；

  • 4.2回到gitbash中，配置github账户信息，两条命令：git config –blobal user.name “liuyongqian2010” ,git config –global user.email “1029853206@qq.com”

  • 4.3在gitbash中创建SSH，输入”ssh-keygen -t rsa -C “1029853206@qq.com”生成ssh，输入”cd ~/.ssh”，”cat id_rsa.pub”后复制出现的一堆码

  • 4.4在github网页端，你的头像下setting中的SSH and GPG keys一栏添加一个New SSH key后在gitbash中输入”ssh -T git@github.com”验证是否添加成功

##5.本地验证

  • 5.1先在gitbash中运用”npm i hexo-server”安装hexo的服务器

  • 5.2使用随便一个编辑器修改blog文件夹下的”_config.yml”文件中的”deploy”模块中的”repo”为你在github上的库地址如”https://github.com/liuyongqian2010/liuyongqian2010.github.io“

  • 5.3在gitbash中cd到blog目录下，分别执行以下三条命令：”hexo clean””hexo generate””hexo server”后打开浏览器输入”http://localhost:4000"可见hexo默认主题页面

##6.上传github

  • 6.1为了将以后所写文章部署到github服务器上让别人看到请在gtibash中输入”npm install hexo-deployer-git –save”

  • 6.2在gitbash中依次执行那三条命令后在浏览器中输入”https://liuyongqian2010.github.io/"可见自己的博客了

##7.添加RSS

  • 7.1在gitbash中输入”npm i hexo-generator-feed”安装rss插件

  • 7.2在blog文件夹下的_config.yml中找到Extension添加如下内容：
    Extensions
    plugin:
    -hexo-generator-feed
    Feed Atom
    feed:

    type: atom
path: atom.xml
limit: 20

##8.写文章了

  • 在gitbash中运用”hexo new ‘文章名’”后可在source/_posts路径下看到你创建的内容了，可下载Markdown进行内容编辑，之后执行那三条命令……

##9.遇到障碍

  • 9.1在window10上运行Markdown可能会总弹出“HTML渲染错误”，下载安装（awesomium_v1.6.6_sdk_win.exe）后重启Markdown即可

  • 9.2详细内容可参见：hexo从零开始到搭建完整
