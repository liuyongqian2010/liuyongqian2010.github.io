元组、字典

元组、字典

元组

  • 定义：元组与列表类似，但是元组的元素不能编辑;元组中只包括一个元素时，需要在元素后加逗号,否则括号会被当作运算符使用

    t1=(2018,'New',True,[1,2])
print(len(t1))        #result:4

  • 访问，使用切片

    t3=('a','b','c','d','e')
print(t3[:])             #result:('a', 'b', 'c', 'd', 'e')
print(t3[0:3])           #result:('a', 'b', 'c')
print(t3[::-1])          #result:('e', 'd', 'c', 'b', 'a')
print(t3[0:4:2])         #result:('a', 'c')

  • 运算符 + * in not in

    t4=(1,2,3)
t5=('a','b','c')
t6=t4+t5                 #新的元组
print(t6)                #result:(1, 2, 3, 'a', 'b', 'c')
print(t6[2:4]*2)         #result:(3, 'a', 3, 'a')
print('a' in t6)         #result:True
print('a' not in t6)     #result:False

  • 内置函数

    #len()
t1=('a','b','c','d','e')
print(len(t1))           #result:5

#max()、min()
t1=('a','b','c','d','e')
print(max(t1))           #result:e
print(min(t1))           #result:a

#tuple()将其他类型转换成元组
list=['a','b','c','d','e']
print(tuple(list))       #result:('a', 'b', 'c', 'd', 'e')

字典

  • 定义：另一种可变容器模型，且可存放任意类型对象，键值对形式，以逗号分隔，键必须是唯一的

    d={'name':'Liu','age':'26','birth':12.22}

  • 增删改查

    #增加
d={'name':'Liu','age':'26','birth':12.22}
d['job']='Engineer'
print(d)          #result:{'name': 'Liu', 'job': 'Engineer', 'age': '26', 'birth': 12.22}

#删除
del d['birth']    #del d["已存d在的键"]
print(d)          #result:{'name': 'Liu', 'job': 'Engineer', 'age': '26'}

#修改
d['age']='108'    #d["已存d在的键"]
print(d)          #result:{'name': 'Liu', 'job': 'Engineer', 'age': '108'}

#查询
d={'name':'Liu','age':'26','birth':12.22}
print(d['name'])                    #result:Liu
print(d.get('name'))                #result:Liu
print(d.get('job','Not find'))      #result:Not find

  • 函数

    #len()
d={'name':'Liu','age':'26','birth':12.22}
print(len(d))           #result:3

#keys()获取键
for k in d.keys():
    print(k,end="\t")   #result:name   age    birth

#values()获取值    
for v in d.values():
    print(v,end="\t")   #result:Liu    26    12.22

#items()获取每个键值对的元组
d={'name':'Liu','age':'26','birth':12.22}
for k,v in d.items():
    print(k,v,end="\t") #result:name Liu   age 26    birth 12.22

#str()转换成字符型
d={'name':'Liu','age':'26','birth':12.22}
s=str(d)
print(type(s))          #result:<class 'str'>

#clear()清空字典键值
d={'name':'Liu','age':'26','birth':12.22}
d.clear()
print(d)                 #result:{}

#copy()复制字典内容
d={'name':'Liu','age':'26','birth':12.22}
dd=d.copy()
print(dd)                #result:{'name': 'Liu', 'birth': 12.22, 'age': '26'}

#pop()没有key会抛出异常
d={'name': 'Liu', 'birth': 12.22, 'age': '26'}
d.pop('age')
print(d)                 #result:{'birth': 12.22, 'name': 'Liu'}

#popitem()随机返回并删除字典中的一对键和值
d={'name': 'Liu', 'birth': 12.22, 'age': '26'}
d.popitem()
print(d)                 #result:{'name': 'Liu', 'birth': 12.22}

列表

列表

列表的定义

  • 列表的数据项不需要具有相同的类型，逗号分隔，方括号括起来

    list1=['News','Git',2018,1.15,False]
print(list1[2])             #result:2018

list=[['a','b'],['A','B'],['a','B']] 
print(list[1][1])           #二维列表  #result:B

列表的切片

  • 列表也可以使用切片，使用方法与字符串一致

    print(list1[:])  #所有       #result:['News', 'Git', 2018, 1.15, False]
print(list1[1:2])           #result:['Git']
print(list1[1:4:2])         #result:['Git', 1.15]
print(list1[-1])            #result:False
print(list1[-1:-5:-1])      #result:[False, 1.15, 2018, 'Git']
print(list1[::-1])             #result:[False, 1.15, 2018, 'Git', 'News']

列表的编辑

  • 增加

    list2=['Happy','New','Year']
list2.append('2018')          #增加到列表尾部
print(list2)      #result:['Happy', 'New', 'Year', '2018']
list2.insert(3,'2018')        #插入到列表相应下标位置,超出范围则在尾部添加
print(list2)      #result:['Happy', 'New', 'Year', '2018']

list3=['百度','小米','阿里']
list4=['李桑','雷军','马云'] #两个列表可以进行相加 
print(list3+list4)            #result:['百度', '小米', '阿里', '李桑', '雷军', '马云']
print(list3)                  #result:['百度', '小米', '阿里']

list3.extend(list4)           #列表1.extend(列表2)，是把列表2迭代后追加到列表1，extend()没有返回值
print(list3)                  #result:['百度', '小米', '阿里', '李桑', '雷军', '马云']

  • 修改

    list5=['百度','小米','阿里']
list5[1]='大米'
print(list5)                  #result:['百度', '大米', '阿里']

  • 删除

    list6=['百度','小米','阿里']
del list6[1]                  #del
print(list6)                  #result:['百度', '阿里']

list7=['百度','小米','阿里']
list7.pop()                   #默认删除最后一个,也可以指定删除的下标，返回删除值
print(list7)                  #result:['百度', '小米']

list8=['百度','小米','阿里']
list8.remove('小米')          #删除括号中的内容，但只删除一次
print(list8)                  #result:['百度', '阿里']

  • 查询

    list9=['a','b','c','d','e']
print(list9.index('c'))       #返回查找到的下标  #result:2
print('a' in list9)            #返回bool值       #result:True
print('a' not in list9)        #返回bool值       #result:False

列表的函数

  • len() 列表元素个数

    list1=['a','b','c','d','e']
print(len(list1))           #result:5

  • max()、min() 返回列表元素最大值、最小值;字母以ASCII为依据

    print(max(list1))           #result:e
print(min(list1))           #result:a

  • list()将元组转换成列表

    a=('a','b','c')             #定义一个元组
print(list(a))              #result:['a', 'b', 'c']

  • append()增加到列表尾部

  • insert(index,obj)插入到列表相应下标位置,超出范围则在尾部添加
  • extend()列表1.extend(列表2)，是把列表2迭代后追加到列表1
  • pop()默认删除最后一个,也可以指定删除的下标，返回删除值
  • remove()删除括号中的内容，但只删除一次

  • reverse()反向列表中的元素

    list2=['a','b','c','d','e']
list2.reverse()
print(list2)                  #result:['e', 'd', 'c', 'b', 'a']

  • sort([func])排序，默认升序

    list3=[5,6,4,2,9]
list3.sort()
print(list3)                  #result:[2, 4, 5, 6, 9]

  • clear() 清空列表

    list4=['a','b','c','d','e']
list4.clear()
print(list4)                  #result:[]

  • copy()复制列表，在内存中又开辟了一个空间；通过等号复制副本，在内存中指向原来的地址

    list5=['a','b','c']
list6=list5.copy()
print(list6)                  #result:['a','b','c']

  • eval()将元组、字典、列表类型的字符串转为元组、字典、列表,可以把list,tuple,dict和string相互转化

    a="[[1,2], [3,4], [5,6]]"        #字符串转换成列表
m=eval(a)
print(type(m))                  #result:<class 'list'>
print(m)                        #result:[[1, 2], [3, 4], [5, 6]]

a="{'name':'Liu','age':'25'}"   #字符串转换成字典
m=eval(a)
print(type(m))                  #result:<class 'dict'>
print(m)                        #result:{'name': 'Liu', 'age': '25'}

a="('a','b','c')"               #字符串转换成元组
m=eval(a)
print(type(m))                  #result:<class 'tuple'>
print(m)                        #result:('a', 'b', 'c')

字符串的定义、切片、格式化、函数

字符串的定义、切片、格式化、函数

定义

  • 单引号、双引号都可以定义字符串，三引号允许一个字符串跨多行

    name1='Honey'  #单引号
name2="Tom"    #双引号
html='''       #三引号
<html>
<head></head>
<body>
    <p>你好，2018！</p>
</body>
</html>
'''

切片

  • 可以使用中括号和下标来获取单个字符或截取字符串，格式为[]、[ : ]、[ : : ]

    str1='liuyongqian2018@163.com' #先定义一个字符串
#正序
a=str1[0] #获取单个字符，取第一个字符      #result:l
b=str1[len(str1)-1] #取最后一个字符，也可以写成 b=str1[-1],python特有         #result:m
c=str1[1:5]#获取字符串，取下标为1、2、3、4的字符，不包括最后一位5      #result:iuyo
d=str1[:]#取所有     #result:liuyongqian2018@163.com
e=str1[1:]#取从下标为1到最后一个字符     #result:iuyongqian2018@163.com
f=str1[:5]#取下标为0到4的字符，不包括5         #result:liuy
g=str1[-5:]#取从-5到结尾的字符     #result:3.com
h=str1[-5:-2]#取从-5到-3的字符，不包括-2         #result:3.c
#逆序
i=str1[ : : ]#取所有，最后一个参数为步长，默认为1     #result:liuyongqian2018@163.com
j=str1[-1:-7:-2]#取-1到-6的字符，步长为2         #result:mc3
k=str1[-1: :-3]#取-1到结尾的字符，步长为3         #result:m.11nqoi
l=str1[-1: :-1]#倒序     #result:moc.361@8102naiqgnoyuil
m=str1[ : :-1]#倒序         #result:moc.361@8102naiqgnoyuil

格式化

  • 第一种，%s 字符，%d 整数，%f 浮点型，%.2f 保留2位小数且四舍五入

  • 第二种，.format()格式化,如果以name开头，其后的变量均需以键值的形式出现

    x='name:{0} age:{1}'.format('Tom',33)     #result: name:Tom age:33
y='name:{name} age:{age}'.format(name="Tom",age=25)      #result: name:Tom age:25
z1='name={0}'.format("Tom")      #result: name=Tom
z2='name={{0}}'.format("Tom")       #result: name={0}
z3='name={{{0}}}'.format("Tom")           #result: name={Tom}

  • 小练习：以上述两种方式输出结果

    name=input("请输入NAME：") #字符型
print("NAME:%s" %name)
age=int(input("请输入AGE："))#整型
print("AGE:%d" %age)

print("WORK:%f"%work)

函数

  • len()字符串的长度

    str3="aabcd efghi jklmn" #先定义一个字符
print('str3的长度为{0}'.format(len(str3)))        #result：str3的长度为17

  • count(str,start,end)返回指定范围内字符出现数量，范围不包括end

    print("str3中a出现次数为{0}".format(str3.count('a',0,2)))     #result：str3中a出现次数为2

  • capitalize()将字符串的第一个字符转换为大写

    print(str3.capitalize())     #result：Aabcd efghi jklmn

  • title()将字符串中每一个单词首字母大写

    print(str3.title())     #result：Aabcd Efghi Jklmn

  • find()返回找到的字符串的下标，没有找到则返回-1

    print('d在str3中的下标为{0}'.format(str3.find('d',1)))#表示第一次出现字符d的下标，        #result：d在str3中的下标为4

  • rfind(str,start,end)从右边开始找,指定一个范围

Ajax回顾

Ajax回顾

简介


Asynchronous Javascript And XML，异步的JavaScript和XML。通过在后台与服务器进行少量数据交换，AJAX可以使网页实现异步更新。即在不重新加载整个网页的情况下，对网页的某部分进行更新。

以封装一个请求为例

function ajax(url,funSucc,funFail){
    //1.创建ajax对象
    var xhr=new XMLHttpRequest()
    ////兼容IE6，IE5
    if(window.XMLHttpRequest){
        var xhr=new XMLHttpRequest()
    }else{
        var xhr=new ActiveXObject("Microsoft.XMLHttp")    
    }
    //2.连接服务器，open(请求类型,文件路径,是否异步)
    //如果是post请求，则需要手动设置请求消息头，如下
    xhr.open('post',url,true);
    xhr.setRequestHeader("Content-type","application/x-www-form-urlencoded");
    //如果是get请求，如下
    xhr.open('get',url,true)
    //3.发送请求，send方法内传递的是提交参数，没有可为null
    xhr.send(null)
    //4.接收数据，为Ajax对象的onreadystatechange事件设定响应函数
    xhr.onreadystatechange=function(){
        //readyState 请求状态:
            0尚未初始化
            1正在发送请求
            2请求完成
            3请求成功，正在接受数据
            4接收数据成功
        if(xhr.readyState==4){
            //status 服务器返回的http请求响应值常用的有：
                200表示请求成功
                202请求被接受但处理未完成
                400错误的请求
                404资源未找到
                500内部服务器错误

            if(xhr.status==200){
                //responseText 服务器返回的文本
                funSucc(xhr.responseText)
            }else{
                //statusText 服务器返回的请求响应文本
                funFail(xhr.statusText)
            }
        }
    }
}

进程与线程

进程与线程

在刚开始学习性能工具LR时，常听到进程与线程；但不得具体是什么意思，或者比较抽象；然后在网络上找到了比较形象的解释

  1. 计算机的核心是CPU，它承担了所有的计算任务。它就像一座工厂，时刻在运行。
  2. 假定工厂的电力有限，一次只能供给一个车间使用。也就是说，一个车间开工的时候，其他车间都必须停工。背后的含义就是，单个CPU一次只能运行一个任务。
  3. 进程就好比工厂的车间，它代表CPU所能处理的单个任务。任一时刻，CPU总是运行一个进程，其他进程处于非运行状态。
  4. 一个车间里，可以有很多工人。他们协同完成一个任务。
  5. 线程就好比车间里的工人。一个进程可以包括多个线程。
  6. 车间的空间是工人们共享的，比如许多房间是每个工人都可以进出的。这象征一个进程的内存空间是共享的，每个线程都可以使用这些共享内存。
  7. 可是，每间房间的大小不同，有些房间最多只能容纳一个人，比如厕所。里面有人的时候，其他人就不能进去了。这代表一个线程使用某些共享内存时，其他线程必须等它结束，才能使用这一块内存。
  8. 一个防止他人进入的简单方法，就是门口加一把锁。先到的人锁上门，后到的人看到上锁，就在门口排队，等锁打开再进去。这就叫”互斥锁”（Mutual exclusion，缩写 Mutex），防止多个线程同时读写某一块内存区域。
  9. 还有些房间，可以同时容纳n个人，比如厨房。也就是说，如果人数大于n，多出来的人只能在外面等着。这好比某些内存区域，只能供给固定数目的线程使用。
  10. 这时的解决方法，就是在门口挂n把钥匙。进去的人就取一把钥匙，出来时再把钥匙挂回原处。后到的人发现钥匙架空了，就知道必须在门口排队等着了。这种做法叫做”信号量”（Semaphore），用来保证多个线程不会互相冲突。

    不难看出，mutex是semaphore的一种特殊情况（n=1时）。也就是说，完全可以用后者替代前者。但是，因为mutex较为简单，且效率高，所以在必须保证资源独占的情况下，还是采用这种设计。
  • 操作系统的设计，因此可以归结为三点：

    （1）以多进程形式，允许多个任务同时运行；

    （2）以多线程形式，允许单个任务分成不同的部分运行；

    （3）提供协调机制，一方面防止进程之间和线程之间产生冲突，另一方面允许进程之间和线程之间共享资源。

1.yum简介

yum的理念是使用一个中心仓库(repository)管理一部分甚至一个distribution 的应用程序相互关系，根据计算出来的软件依赖关系进行相关的升级、安装、删除等等操作，减少了Linux 用户一直头痛的dependencies 的问题。这一点上，yum 和apt 相同。apt 原为debian 的deb 类型软件管理所使用，但是现在也能用到RedHat门下的rpm 了。

yum 主要功能是更方便的添加/删除/更新RPM 包，自动解决包的倚赖性问题，便于管理大量系统的更新问题。

yum 可以同时配置多个资源库(Repository)，简洁的配置文件（/etc/yum.conf），自动解决增加或删除rpm 包时遇到的依赖性问题，保持与RPM 数据库的一致性。

# CentOS-Media.repo
#
# This repo is used to mount the default locations for a CDROM / DVD on
#  CentOS-5.  You can use this repo and yum to install items directly off the
#  DVD ISO that we release.
#
# To use this repo, put in your DVD and use it with the other repos too:
#  yum --enablerepo=c5-media [command]
#  
# or for ONLY the media repo, do this:
#
#  yum --disablerepo=\* --enablerepo=c5-media [command]     
[c5-media]
name=CentOS-$releasever - Media
baseurl=file:///media/CentOS/
        file:///mnt/cdrom/
        file:///media/cdrecorder/
gpgcheck=1
enabled=1
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-5

–在baseurl中修改第2个路径为/mnt/cdrom（即为光盘挂载点）将enabled=0改为1

–禁用默认的yum 网络源mv CentOs-Base.repo CentOs-Base.repo.bak

–运行yum命令yum install postgresql

2.5我们就用系统默认的yum源，速度也还是可以的！
