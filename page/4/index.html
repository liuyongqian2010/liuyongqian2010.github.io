Python3.5连接MySQl数据库

PyMySQL是在Pyhton3.x版本中用于连接MySQL数据库的一个库，Python2中则使用mysqldb库

使用pip3 install PyMySQl进行安装PyMySQL库


数据库连接

连接数据库前，请先确认以下事项：

  • 已经创建了数据库 TESTDB

  • 在TESTDB数据库中您已经创建了表 t_table

  • t_table表字段为 FIRST_NAME, LAST_NAME, AGE, SEX 和 INCOME

  • 连接数据库testdb使用的用户名为 “testuser” ，密码为 “test123”，你可以自己设定或者直接使用root用户名及其密码

  • mysql数据库用户授权请使用grant命令

以下示例是连接MySQL数据库中的TESTDB数据库：

#!/usr/bin/python3

import pymysql
# 打开数据库连接
db = pymysql.connect("localhost","testuser","test123","TESTDB" )
# 使用 cursor() 方法创建一个游标对象 cursor
cursor = db.cursor()
# 使用 execute()  方法执行 SQL 查询 
cursor.execute("SELECT VERSION()")
# 使用 fetchone() 方法获取单条数据.
data = cursor.fetchone()
print ("Database version : %s " % data)
# 关闭数据库连接
db.close()

操作实例

创建数据库表

如果数据库连接存在，可以使用execute()方法为数据库创建表，如下所示创建表t_table

#!/usr/bin/python3

import pymysql
# 打开数据库连接
db = pymysql.connect("localhost","testuser","test123","TESTDB" )
# 使用 cursor() 方法创建一个游标对象 cursor
cursor = db.cursor()
# 使用 execute() 方法执行 SQL，如果表存在则删除
cursor.execute("DROP TABLE IF EXISTS t_table")
# 使用预处理语句创建表
sql = """CREATE TABLE t_table(
         FIRST_NAME  CHAR(20) NOT NULL,
         LAST_NAME  CHAR(20),
         AGE INT,  
         SEX CHAR(1),
         INCOME FLOAT )"""
cursor.execute(sql)
# 关闭数据库连接
db.close()
数据库插入操作

以下示例使用执行SQL insert 语句 向表t_table中插入一条记录：

#!/usr/bin/python3

import pymysql
# 打开数据库连接
db = pymysql.connect("localhost","testuser","test123","TESTDB" )
# 使用cursor()方法获取操作游标 
cursor = db.cursor()
# SQL 插入语句
sql = """INSERT INTO t_table(FIRST_NAME,
         LAST_NAME, AGE, SEX, INCOME)
         VALUES ('Mac', 'Mohan', 20, 'M', 2000)"""
try:
   # 执行sql语句
   cursor.execute(sql)
   # 提交到数据库执行
   db.commit()
except:
   # 如果发生错误则回滚
   db.rollback()
# 关闭数据库连接
db.close()

执行事务

事务机制可以确保数据一致性。

事务应该具有4个属性：原子性、一致性、隔离性、持久性,这四个属性通常称为ACID特性。

- 原子性（atomicity）。一个事务是一个不可分割的工作单位，事务中包括的诸操作要么都做，要么都不做

- 一致性（consistency）。事务必须是使数据库从一个一致性状态变到另一个一致性状态。一致性与原子性是密切相关的

- 隔离性（isolation）。一个事务的执行不能被其他事务干扰。即一个事务内部的操作及使用的数据对并发的其他事务是隔离的，并发执行的各个事务之间不能互相干扰

- 持久性（durability）。持续性也称永久性（permanence），指一个事务一旦提交，它对数据库中数据的改变就应该是永久性的。接下来的其他操作或故障不应该对其有任何影响。

Python DB API 2.0 的事务提供了两个方法 commit 或 rollback

# SQL删除记录语句
sql = "DELETE FROM EMPLOYEE WHERE AGE > '%d'" % (20)
try:
   # 执行SQL语句
   cursor.execute(sql)
   # 向数据库提交
   db.commit()
except:
   # 发生错误时回滚
   db.rollback()

对于支持事务的数据库， 在Python数据库编程中，当游标建立之时，就自动开始了一个隐形的数据库事务。

commit()方法游标的所有更新操作，rollback（）方法回滚当前游标的所有操作。每一个方法都开始了一个新的事务。

2017_12_16一台PC机同时安装Python2与python3

由于平时学习中需要两种版本不时的替换，索性通过网上查找资料自己配置安装了两个版本，以下是小白整理的一些步骤。

Python2安装

  • 1.1到Python官网下载Python2安装文件，如：python-2.7.12.amd64.exe
  • 1.2点击”python-2.7.12.amd64”进行安装后将该安装目录下的python.exe文件重新命名为python2.exe
  • 1.3在电脑–属性–高级系统设置–高级–环境变量–系统变量–Path–编辑中添加环境变量，如”C:\Python27”，即python2的安装目录
  • 1.4在Dos窗口中输入python2如出现python2的版本信息即表明安装成功
  • 1.5到Pip官网下载名称类似”setuptools-28.7.0.zip”的pip依赖包，在cmd中cd到该依赖包解压后的目录下，输入python2 setup.py install
  • 1.6到Pip官网下载名称类似”pip-8.1.2.tar”的文件并解压，在cmd中cd到该文件解压后的目录下，再次输入python2 setup.py install
  • 1.7在cmd中输入pip2,如果出现”Usage”字样说明安装成功
  • 1.8测试使用：在cmd中输入pip2 install selenium,如果窗口命令哗哗的下滚，说明可以了；也可下载python其他包，用cmd进入到其解压后的目录下，输入python2 setup.py install

Python3安装

  • 1.1到Python官网下载Python3安装文件，如：python-3.5.0-amd64.exe
  • 1.2点击”python-3.5.0-amd64.exe”进行安装后将该安装目录下的python.exe文件重新命名为python2.exe
  • 1.3在电脑–属性–高级系统设置–高级–环境变量–系统变量–Path–编辑中添加环境变量，如”C:\Python35”，即python3的安装目录
  • 1.4在Dos窗口中输入python如出现python3.5的版本信息即表明安装成功；由于未修改python3安装目录下的python.exe文件名，此处python就是python3
  • 1.5在cmd中输入pip3,如果出现”Usage”字样说明自动安装成功；python3自带pip，无需手动安装
  • 1.6测试使用：在cmd中输入pip3 install selenium,如果窗口命令哗哗的下滚，说明可以了；也可下载python其他包，用cmd进入到其解压后的目录下，输入python setup.py install
  • 该文参考大神的佳作在同一台电脑上同时安装Python2和Python3
  • 若文中内容有误，希望大家指正，谢谢!

Github+Hexo博客更换主题

1.挑选主题

  • 1.1在Themes|Hexo中挑选自己中意的主题，记住名字，比如“Again”
  • 1.2在gitbash中cd到博客所在文件夹的位置如”cd /e/github/myblog”,继续输入”https://github.com/lyyourc/hexo-theme-again"可克隆最新的Again主题到你的博客文件夹里面
  • 1.3克隆之后可在你的博客文件中的themes下看到刚刚下载的Again主题

2.安装主题

  • 2.1修改站点配置文件（你的博客最底层目录下的_config.yml），用任一编辑器打开找到“theme”选项，将原先的主题名称删除改为Again即可，注意theme的冒号与Again之间有空格

  • 2.1.1这里可以按照Again的官方指导教程，配置一下Site模块中的内容：

  • 2.2在gitbash中输入”hexo s –debug”启动hexo服务器进行本地测试一下，在浏览器打开“http://localhost:4000/”

  • 2.3修改主题配置文件（你的博客目录下的themes/Again下的_config.yml）,找到scheme关键字，保证只有一个scheme前面没有#号，说明这个scheme选中

3.发布主题

  • 在gitbash中依次输入”hexo clean” “hexo generate” “hexo deploy”，字幕滚动完后，新博客就可以访问了

4.寻找大神

2017_12_15 Git+NodeJS+Hexo创建个人博客

##1.Git篇

  • Windows环境下，安装（Git-2.15.1.2-64-bit.exe）后可打开gitbash运用”git version”查看版本

##2.NodeJS篇

  • Windows环境下，安装（node-v8.9.3-x64.msi）并配置环境变量后，在win-dos窗口中运用”node -v”查看版本

##3.Hexo篇（接下来均在gitbash中完成

  • 3.1手动创建一个将来存放blog的文件夹，在gitbash中cd到该文件夹下运用”npm i -g hexo”安装hexo后使用”hexo -v”查看版本；

  • 3.2运用”hexo init”初始化文件夹

##4.关联github

  • 4.1在github网页端创建自己的账号后new一个repository，name如liuyongqian2010.github.io，记得勾选initialize；

  • 4.2回到gitbash中，配置github账户信息，两条命令：git config –blobal user.name “liuyongqian2010” ,git config –global user.email “1029853206@qq.com”

  • 4.3在gitbash中创建SSH，输入”ssh-keygen -t rsa -C “1029853206@qq.com”生成ssh，输入”cd ~/.ssh”，”cat id_rsa.pub”后复制出现的一堆码

  • 4.4在github网页端，你的头像下setting中的SSH and GPG keys一栏添加一个New SSH key后在gitbash中输入”ssh -T git@github.com”验证是否添加成功

##5.本地验证

  • 5.1先在gitbash中运用”npm i hexo-server”安装hexo的服务器

  • 5.2使用随便一个编辑器修改blog文件夹下的”_config.yml”文件中的”deploy”模块中的”repo”为你在github上的库地址如”https://github.com/liuyongqian2010/liuyongqian2010.github.io“

  • 5.3在gitbash中cd到blog目录下，分别执行以下三条命令：”hexo clean””hexo generate””hexo server”后打开浏览器输入”http://localhost:4000"可见hexo默认主题页面

##6.上传github

  • 6.1为了将以后所写文章部署到github服务器上让别人看到请在gtibash中输入”npm install hexo-deployer-git –save”

  • 6.2在gitbash中依次执行那三条命令后在浏览器中输入”https://liuyongqian2010.github.io/"可见自己的博客了

##7.添加RSS

  • 7.1在gitbash中输入”npm i hexo-generator-feed”安装rss插件

  • 7.2在blog文件夹下的_config.yml中找到Extension添加如下内容：
    Extensions
    plugin:
    -hexo-generator-feed
    Feed Atom
    feed:

    type: atom
path: atom.xml
limit: 20

##8.写文章了

  • 在gitbash中运用”hexo new ‘文章名’”后可在source/_posts路径下看到你创建的内容了，可下载Markdown进行内容编辑，之后执行那三条命令……

##9.遇到障碍

  • 9.1在window10上运行Markdown可能会总弹出“HTML渲染错误”，下载安装（awesomium_v1.6.6_sdk_win.exe）后重启Markdown即可

  • 9.2详细内容可参见：hexo从零开始到搭建完整
