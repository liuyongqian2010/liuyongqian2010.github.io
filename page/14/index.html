封装、继承、多态

封装、继承、多态

封装

  • 隐藏内部实现，提供外部接口
    两个步骤：属性私有化——set\get函数写方法

    class Student:
    def __init__(self,name,age):
        self.name=name
        self.__age=age
    @property
    def age(self):
        return self.__age
    @age.setter
    def age(self,value):
        if value<0 or value>100:
            self.__age=18
        else:
            self.__age=value
m=Student('刘桑',25)
m.age=100
print(m.age)              #result:100

继承

  • 继承：子类继承父类，子类可以使用父类的属性和方法，简化代码
    子类继承父类，父类派生子类（父类又叫基类，子类又叫派生类）
    继承指出两个类符合“A是B”的关系

    class Vehicle:         #交通工具类
    def __init__(self,name):
        self.name=name
        print('交通工具被初始化！')
    def run(self):
        print('交通工具在行驶...')
    def show(self):
        print('车的名字是%s'%self.name)
class Truck(Vehicle):    #卡车，属性：载重
    #当子类继承父类时，子类的构造方法应该包含父类和子类共有的属性，如name
    #在子类的初始化方法中，将父类的属性传递给父类，子类的属性赋值给子类
    #在子类的初始化方法中，调用父类的属性有四种方法：
    # 1，父类名.属性 2，self.属性 3，父类名.__init__(self,属性) 4，super(子类名,self).__init__(属性)
    def __init__(self,name,weight):
        #Vehicle.name=name                 #1，父类名.属性
        #self.name=name                    #2，self.属性
        #Vehicle.__init__(self,name)       #3，父类名.__init__(self,属性)
        super(Truck,self).__init__(name)   #4，super(子类名,self).__init__(属性)
        self.weight=weight
        print('卡车被初始化！')
    def run(self):        #卡车，重写行驶方法
        #super().show() #调用父类方法
        #Vehicle.show(self)
        super(Truck,self).show()
        print('%s卡车载重%d吨在行驶...'%(self.name,self.weight))
class Train(Vehicle):     #火车，属性：车厢个数
    def __init__(self,name,nums):
        self.nums=nums
    def run(self):        #火车，重写行驶方法
        print('火车在轨道行驶...')

t=Truck('解放',25)
t.run()

  • 三代继承：子类初始化方法需祖父、父类及自己的属性，可以调用父类的初始化方法传参，可以重写父类的方法，构造的顺序依然先构造祖父类，再构造父类，最后构造自己

  • 多继承：一个类同时继承多个父类，class(A,B),当AB均有相同方法，而子类又重写时，调用子类的方法；如果子类没有方法，则调用优先继承的类

    多态
  • 封装可以隐藏实现细节，使得代码模块化；继承可以扩展已存在的代码模块（类）；它们的目的都是为了——代码重用。而多态则是为了实现另一个目的——接口重用！多态的作用，就是为了类在继承和派生的时候，保证使用“家谱”中任一类的实例的某一属性时的正确调用

  • 多态就是多种表现形态的意思。它是一种机制、一种能力，而非某个关键字。它在类的继承中得以实现，在类的方法调用中得以体现。多态意味着变量并不知道引用的对象是什么，根据引用对象的不同表现不同的行为方式

类与对象

基础定义

  • 在python中，一个对象的特征也称为属性（attribute），它所具有的行为也称为方法（method），把具有相同属性和方法的对象归为一个类（class）。即类就是具有相同属性和方法的对象的集合

  • 类名规范：首字母大写，以大写字母分割单词，不使用下划线;不允许使用数字、关键字、除下划线以外的特殊字符开头

    属性名规范：使用名词，首字母小写，以大写字母分割单词

    方法名规范：使用谓词（动词+对象），首字母小写，以大写字母分割单词

  • 注解@property的使用
    它的作用是将一个函数作为类的一个属性来访问，或者把一个getter方法变成属性，@属性名.setter把setter方法变成属性赋值

    class Student:
    def __init__(self,name,age):
        self.name=name
        self.__age=age
    #先定义两个函数，函数名为属性名，第一个函数返回属性值，第二个函数修改属性值

    def age(self):
        return self.__age

    def age(self,value):
        if value<0 or value>100:
            self.__age=18   #给一个默认值
        else:
            self.__age=value

函数

定义
  • 高复用、低耦合

  • 语法

    def add(a,b):        #def 函数名(参数列表):
                     #函数体
    return a+b       #可以返回多个值，返回的多个值组成元组；返回值加上中括号，返回列表      
ret=add(1,2)
print('返回值',ret)           #result:返回值 3

  • 可更改与不可更改对象

    def fun(a):
    a=10
    print('函数内部a为',a)
a=5
print('调用前',a)      #result:调用前 5
fun(a)                #result:函数内部a为 10
print('调用后',a)      #result:调用后 5
#在python中，string,tuples,numbers是不可更改的对象，list,dict则是可以修改的对象
#不可变类型：变量赋值a=5后再次赋值a=10，这里实际是新生成一个int值对象10，再让a指向它，而5被丢弃，不是改变a的值；相当于新生成了a
#不可变类型：值传递（复制副本），如fun(a),传递的只是a的值，没有影响a对象本身；如在fun(a)内部修改a的值，只是修改另一个复制的对象，不会影响a本身

def fun(b):
    b[0]='刘备'
    b[1]='刘禅'
b=['刘邦','刘秀']
print('调用前',b)      #result:调用前 ['刘邦', '刘秀']
fun(b)
print('调用后',b)      #result:调用后 ['刘备', '刘禅']
#可变类型：变量赋值b=[1,2,3]后再次赋值b[2]=5则是将b的第三个元素值更改，本身b没有动，只是其内部的一部分值被修改了
#可变类型：引用传递(实质是值传递，传递可变类型-可以改变内容，但不可以改变原地址)，如fun(b)，则是将b真正的传过去，修改后fun外部的b也会受影响

def fun(b):
    b=['刘备','刘禅']
    print('函数内部b为',b)
b=['刘邦','刘秀']
print('调用前',b)      #result:调用前 ['刘邦', '刘秀']
fun(b)                   #result:函数内部b为 ['刘备', '刘禅']
print('调用后',b)      #result:调用后 ['刘邦', '刘秀']
#相当于不可变类型
参数类型
  • 在Python中定义函数,可用必选参数、默认参数、可变参数、关键字参数、命名关键字参数；除了可变参数无法和命名关键字参数组合使用外，其他均可组合使用；参数定义的顺序必须是：必选参数、默认参数、可变参数/命名关键字参数和关键字参数

  • 必选参数（形参与实参的个数、类型、顺序一一对应）

    def fun1(a,b):              #形参：定义方法
    '''
    :param a:               第一个加数
    :param b:               第二个加数
    :return:                和
    '''
    return a+b

print(fun1.__doc__)         #调看文档
print(h)                    #result:99

  • 默认参数（缺省参数）须写在后面，可以不指定参数名，但是顺序要正确，否则需要指定参数名

    def fun2(a,b=3):
    return a+b

  • 可变参数（不定长参数）该参数允许传入0个或者任意个参数，这些参数在函数调用时组装为一个tuple

    for i in nums:
        sum+=i
    return sum

  • 命名关键字参数 该参数允许传入0个或者任意个参数，这些参数在函数内部自动组装为一个dict;一旦以name=value的的传参数的形式存在，之后都要以这种形式存在，但不要求顺序

字符串的定义、切片、格式化、函数

字符串的定义、切片、格式化、函数

定义

  • 单引号、双引号都可以定义字符串，三引号允许一个字符串跨多行

    name1='Honey'  #单引号
name2="Tom"    #双引号
html='''       #三引号
<html>
<head></head>
<body>
    <p>你好，2018！</p>
</body>
</html>
'''

切片

  • 可以使用中括号和下标来获取单个字符或截取字符串，格式为[]、[ : ]、[ : : ]

    str1='liuyongqian2018@163.com' #先定义一个字符串
#正序
a=str1[0] #获取单个字符，取第一个字符      #result:l
b=str1[len(str1)-1] #取最后一个字符，也可以写成 b=str1[-1],python特有         #result:m
c=str1[1:5]#获取字符串，取下标为1、2、3、4的字符，不包括最后一位5      #result:iuyo
d=str1[:]#取所有     #result:liuyongqian2018@163.com
e=str1[1:]#取从下标为1到最后一个字符     #result:iuyongqian2018@163.com
f=str1[:5]#取下标为0到4的字符，不包括5         #result:liuy
g=str1[-5:]#取从-5到结尾的字符     #result:3.com
h=str1[-5:-2]#取从-5到-3的字符，不包括-2         #result:3.c
#逆序
i=str1[ : : ]#取所有，最后一个参数为步长，默认为1     #result:liuyongqian2018@163.com
j=str1[-1:-7:-2]#取-1到-6的字符，步长为2         #result:mc3
k=str1[-1: :-3]#取-1到结尾的字符，步长为3         #result:m.11nqoi
l=str1[-1: :-1]#倒序     #result:moc.361@8102naiqgnoyuil
m=str1[ : :-1]#倒序         #result:moc.361@8102naiqgnoyuil

格式化

  • 第一种，%s 字符，%d 整数，%f 浮点型，%.2f 保留2位小数且四舍五入

  • 第二种，.format()格式化,如果以name开头，其后的变量均需以键值的形式出现

    x='name:{0} age:{1}'.format('Tom',33)     #result: name:Tom age:33
y='name:{name} age:{age}'.format(name="Tom",age=25)      #result: name:Tom age:25
z1='name={0}'.format("Tom")      #result: name=Tom
z2='name={{0}}'.format("Tom")       #result: name={0}
z3='name={{{0}}}'.format("Tom")           #result: name={Tom}

  • 小练习：以上述两种方式输出结果

    name=input("请输入NAME：") #字符型
print("NAME:%s" %name)
age=int(input("请输入AGE："))#整型

函数

  • len()字符串的长度

    str3="aabcd efghi jklmn" #先定义一个字符
print('str3的长度为{0}'.format(len(str3)))        #result：str3的长度为17

  • count(str,start,end)返回指定范围内字符出现数量，范围不包括end

    print("str3中a出现次数为{0}".format(str3.count('a',0,2)))     #result：str3中a出现次数为2

  • capitalize()将字符串的第一个字符转换为大写

    print(str3.capitalize())     #result：Aabcd efghi jklmn

  • title()将字符串中每一个单词首字母大写

    print(str3.title())     #result：Aabcd Efghi Jklmn

  • find()返回找到的字符串的下标，没有找到则返回-1

    print('d在str3中的下标为{0}'.format(str3.find('d',1)))#表示第一次出现字符d的下标，        #result：d在str3中的下标为4

  • rfind(str,start,end)从右边开始找,指定一个范围

    print('d在str3中的下标为{0}'.format(str3.rfind('d',-18,-1)))        #result：d在str3中的下标为4

  • index()找下标，没有找到则抛出异常

    print('d在str3中的下标为{}'.format(str3.index('d')))         #result：d在str3中的下标为4

  • rindex()从右边开始找，可以指定范围

    print('d在str3中的下标为{}'.format(str3.index('d',-18,-1)))    #result：d在str3中的下标为4

  • replace()替换,但是原字符不会改变

    print("替换str3中的a为{}".format(str3.replace('a','A')))        #result:替换str3中的a:AAbcd efghi jklmn

  • split(str,num)如果括号内什么都不写，就是将通过\t和空格进行拆分，可设置拆分次数

    print("str3默认拆分结果为{0}".format(str3.split()))         #result:str3默认拆分结果为['aabcd', 'efghi', 'jklmn']

  • startswith(str,start,end)检查是否以str开头，可指定检查范围，返回bool值

    print(str3.startswith('aa'))    #result:True

  • endswith()检查是否以str开头，返回bool值

    print(str3.endswith('mn'))        #result:True

  • lower()忽略大小写的地方，全部转为小写

    print(str3.lower())        #result:aabcd efghi jklmn

  • upper()忽略大小写的地方，全部转为大写

    print(str3.upper())        #result:AABCD EFGHI JKLMN

  • ljust(num,str),center(num,str),rjust(num,str)字符串输出时怎样对齐，字符串本来长度是5，把它变成长度为10，可用字符填充

    str4='abcde'#先定义一个字符串，以ljust()为例
print(str4.ljust(10,'*'))        #result:abcde*****

  • lstrip(),strip(),rstrip()去除空格

    str5='   abcde'#先定义一个字符串，以lstrip()为例
print(str4.lstrip())        #result:abcde

  • partition(str)以此字符串为节点拆分,返回头、分割符、尾三部分

    str6='hello python-world'#先定义一个字符串
print(str6.partition("-"))        #result:('hello python', '-', 'world')

  • splitlines(bool keepends)按照换行符进行拆分，如果字符串中有”\n” 结果里是否去掉换行符(‘\r’, ‘\r\n’, \n’)，默认为 False，不包含换行符;如果为True，则保留换行符

    str7='你好\npython-world'#先定义一个字符串
print(str7.splitlines())    #result：['你好', 'python-world']
print(str7.splitlines(True))    #result：['你好\n', 'python-world']

  • isalpha()判断字符串是否全是字母/文字,即非特殊字符，返回bool值

    str8='你好Pythonworld'#先定义一个字符串
print(str8.isalpha())        #result：True

  • isdigit()判断字符串是否是数字，返回bool值

    print(str8.isdigit())        #result：False

  • isalnum()判断是否是数字和字母/文字，返回bool值

    print(str8.isalnum())        #result：True

  • join(sequence)将列表组成新的字符串,sequence为要拼接列表的连接项

    str9=('P','y','t','h','o','n')
str10='-'
print(str10.join(str9))        #result:P-y-t-h-o-n

    email=input("请输入邮箱：")

    print("该邮箱合法！")
    print("用户名部分：{}".format(email[:a]))
else:
    print("该邮箱不合法！")
